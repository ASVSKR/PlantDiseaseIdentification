Plant Disease Classification Project

This is a web application that classifies plant diseases using a pre-trained Convolutional Neural Network (CNN) model. MobileNetV2 has been used to train a CNN over 38 different types of plant based diseases and the model is expected to predict the Disease.

Things acheived in this project are:

  1. DB has been created with multiple classes and related information to the disease
  2. Model has been trained and deployed to identify the disease
